/setup 시작하기

/setup 서버 관리자 전용 공지 채널 · RLNL 게임 · 결과 채널
📦 v1 개요 ⚙ /setup 설정 🎮 RLNL 게임 패널 📅 출석 · 위로금 · 리필

/setup은 라니봇 v1이 이 서버에서 어떻게 동작할지 한 번에 정리하는 관리자용 설정 명령어입니다. 공지 채널, RLNL 게임 기능 ON/OFF, 게임 결과 채널까지 한 번에 지정합니다.

👉 이 명령어는 서버 관리 권한이 있는 관리자만 사용할 수 있습니다.
📦 v1 개요로 돌아가기 🎮 RLNL 게임 패널 보기 📅 출석 · 위로금 · 레벨 보기

1. /setup으로 설정하는 항목

  • 공지 채널 (announce_channel) → 공지 임베드를 보낼 디스코드 채널.
  • RLNL 게임 ON/OFF (enable_game) → 바카라, 출석, 위로금, 리필 기능을 사용할지 여부.
  • 게임 결과 채널 (result_channel, 선택) → 바카라 결과 임베드를 모아볼 채널. 비우면 게임 버튼을 누른 채널로 전송.
기본 예시: /setup announce_channel: #📢-서버-공지 enable_game: true result_channel: #🎴-rlnl-게임-결과

2. 자주 쓰는 패턴

2-1) 공지 + RLNL 게임 같이 쓰는 서버

  • 공지 채널은 따로, RLNL 게임 결과는 전용 채널로 모아서 보고 싶을 때.
  • 추천 설정: 공지 채널 1개, RLNL 결과 채널 1개, enable_game: true.

2-2) 공지만 쓰고 싶은 서버

  • 공지 기능만 쓰고, RLNL 게임은 아예 끄고 싶은 경우.
  • 추천 설정: 공지 채널만 선택, enable_game: false, 결과 채널은 비워둠.

3. /panel · 관리자 패널과 연결

/setup으로 설정을 마친 뒤에는 /panel 명령어로 관리자 패널을 열 수 있습니다. 여기서 공지 모달 버튼과 RLNL 상태를 한눈에 확인할 수 있습니다.

  • ✍ 공지 작성 (모달) 버튼으로 공지 임베드를 바로 생성.
  • 현재 RLNL 게임이 ON/OFF인지, 결과 채널이 어떤 채널인지 표시.
  • 설정을 바꾸고 싶을 때는 다시 /setup을 호출해서 덮어쓰기.

4. RLNL 게임과의 연동

enable_game을 true로 설정해 두면 RLNL 관련 명령어와 패널이 정상 동작합니다. 꺼져 있으면 유저에게 “이 서버에서는 RLNL 게임 기능이 꺼져 있습니다.” 라는 안내가 표시됩니다.

  • /gamepanel – 유저용 RLNL 게임 패널을 띄우는 명령어.
  • RLNL 패널 버튼 – 바카라, 출석, 일일 위로금, 30분 리필 기능 제공.
  • 바카라 결과는 설정된 결과 채널이 있으면 그쪽으로, 없으면 게임을 누른 채널로 전송.

5. 자주 묻는 질문

Q. /setup을 여러 번 써도 되나요?

네. 같은 서버에서 다시 실행하면 기존 설정 위에 덮어쓰게 됩니다. 공지 채널이나 결과 채널을 바꾸고 싶을 때는 그대로 다시 호출하면 됩니다.

Q. 나중에 결과 채널만 따로 설정할 수 있나요?

처음에는 결과 채널을 비워둔 뒤, 나중에 채널을 만든 다음 다시 /setup으로 결과 채널만 지정해도 됩니다.

Q. 게임 기능만 잠깐 끄고 싶어요.

공지 기능은 그대로 두고 RLNL 게임만 끄고 싶다면 enable_game: false로 다시 /setup을 실행하면 됩니다.
