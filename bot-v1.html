/setup 시작하기

/setup 서버 관리자 전용 공지 채널 · RLNL 게임 · 결과 채널

/setup은 배포봇 v1이 서버에서 어떻게 동작할지 한 번에 설정하는 명령어입니다. 공지 채널, RLNL 게임 기능 ON/OFF, 게임 결과를 보낼 채널까지 한 번에 지정할 수 있습니다.

👉 이 명령어는 서버 관리 권한이 있는 관리자만 사용할 수 있습니다.
⚙ /setup 시작하기 🎮 게임 패널 시작하기 📅 출석 기능 시작하기

1. /setup 파라미터 설명

기본 형식

/setup
  announce_channel: #공지-채널
  enable_game: true / false
  result_channel: #게임-결과-채널 (옵션)
  • announce_channel : 공지 임베드를 보낼 디스코드 채널입니다.
  • enable_game : RLNL 바카라/출석/위로금 기능을 켤지 여부입니다.
  • result_channel : 게임 결과 임베드를 모아볼 채널 (비우면 버튼 누른 채널 사용).

실제 코드 동작

내부적으로는 guilds 테이블에 다음 값이 저장됩니다.

  • guild_id – 서버 ID (PRIMARY KEY)
  • announce_channel_id – 공지 채널 ID
  • game_enabled – RLNL 게임 ON(1)/OFF(0)
  • result_channel_id – 결과 채널 ID (nullable)

2. 사용 예시

2-1) 게임까지 함께 사용하는 서버

공지 + RLNL 게임을 모두 사용하고, 게임 결과를 따로 모아보고 싶다면 이렇게 설정할 수 있습니다.

/setup
  announce_channel: #📢-서버-공지
  enable_game: true
  result_channel: #🎴-rlnl-게임-결과
  • 공지 모달로 작성한 공지는 #📢-서버-공지 채널에 전송됩니다.
  • RLNL 바카라 / 출석 / 일일 위로금 / 30분 리필 기능이 활성화됩니다.
  • 바카라 결과 임베드는 항상 #🎴-rlnl-게임-결과 채널로 전송됩니다.

2-2) 공지만 사용하고 싶은 서버

게임 기능 없이 공지 기능만 쓰고 싶은 경우:

/setup
  announce_channel: #📢-서버-공지
  enable_game: false
  • 공지 기능은 정상 동작하지만, RLNL 게임 관련 버튼/명령은 모두 비활성화됩니다.
  • 나중에 게임 기능을 켜고 싶으면 enable_game: true로 다시 설정하면 됩니다.

3. /panel · 관리자 패널

/setup으로 기본 설정을 마쳤다면, 이제 /panel 명령어로 관리자 패널을 열 수 있습니다.

/panel
  • 서버 관리 권한이 있는 관리자만 사용할 수 있습니다.
  • 패널에는 ✍ 공지 작성 (모달) 버튼이 포함됩니다.
  • 현재 RLNL 게임 ON/OFF 상태와 결과 채널 정보도 같이 표시됩니다.

3-1) 공지 작성 (모달)

패널에서 ✍ 공지 작성 (모달) 버튼을 누르면, 아래와 같은 UI가 뜹니다.

  • 공지 제목 – 임베드 타이틀.
  • 공지 내용 – 임베드 본문.

작성 완료 후 제출하면, 코드에서 설정된 announce_channel_id 채널로 임베드 공지가 발송됩니다.

👉 공지 색상/아이콘은 매번 랜덤으로 선택되며, 서버 아이콘이 있으면 썸네일로 자동 첨부됩니다.

4. RLNL 게임과의 연동

/setup에서 enable_game: true로 설정해 두면, 이 서버에서 RLNL 관련 명령어와 패널이 정상 동작합니다.

  • /gamepanel – 유저용 RLNL 게임 패널을 띄움.
  • 패널 버튼 🎮 RLNL 게임 시작 – 바카라 베팅·결과를 내부 DB와 결과 채널에 기록.
  • 패널 버튼 💸 일일 위로금 – 하루 1회 무료 포인트 지급.
  • 패널 버튼 🔁 30분 리필 – 30분 쿨타임 리필 포인트 지급.

이 모든 기능은 guilds 테이블의 game_enabled 값이 1일 때만 활성화되며, 0이면 “이 서버에서는 RLNL 게임 기능이 꺼져 있습니다.” 라는 안내가 표시됩니다.

5. 자주 묻는 질문

Q. /setup 을 여러 번 써도 되나요?

네, 가능합니다. 같은 guild_id로 다시 호출하면 ON CONFLICT(guild_id) DO UPDATE 로 기존 설정을 덮어씁니다.

Q. 결과 채널을 나중에만 설정하고 싶어요.

처음에는 result_channel를 비워두고, 나중에 다시 /setup을 호출해서 채널만 추가해도 됩니다. 설정 전에는 바카라 결과가 버튼을 누른 채널로 전송됩니다.

Q. 게임 기능만 끄고 싶어요.

공지는 유지하고 RLNL 게임만 끄고 싶다면 enable_game: false로 다시 /setup 을 실행하세요.
